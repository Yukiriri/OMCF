
# 测试环境
| JDK        | Liberica LTS 25        |
| :--------- | :--------------------- |
| 服务端核心 | Fabric 1.21.11         |
| 关键MOD    | `C2ME`(配置全核32线程) |

# 负载方式
1. 删除存档中的region文件夹
2. 启动服务端
3. 运行/chunky radius 1024

# log统计
- -Xms2G -Xmx2G
  | 记录项     | G1GC   | ZGC         | SGC            |
  | :--------- | :----- | :---------- | :------------- |
  | chunky用时 | 0m25s  | 0m28s       | 0m31s          |
  | STW时间    | 4~13ms | 0.04~5.56ms | 0.05ms~24.21ms |

- -Xms32G -Xmx32G
  | 记录项     | G1GC   | ZGC         | SGC            |
  | :--------- | :----- | :---------- | :------------- |
  | chunky用时 | 0m26s  | 0m24s       | 0m28s          |
  | STW时间    | 6~12ms | 0.05~0.18ms | 0.08ms~13.05ms |

> [!NOTE]  
> 这个STW时间取自所有Pause阶段里的最高统计  
> 然后取出现的最小值和最大值
